# 🔧 SISTEMA-MANTENIMIENTO

Sistema de gestión de mantenimiento desarrollado en **PHP y MySQL**, diseñado para administrar y controlar el mantenimiento de equipos. Permite registrar dispositivos, programar mantenimientos preventivos y correctivos, así como llevar un seguimiento de incidencias, reparaciones y estado de los equipos.

---

## 🛠 Funcionalidades principales

- ✅ Registro y gestión de equipos
- ✅ Programación de mantenimientos preventivos
- ✅ Registro de mantenimientos correctivos
- ✅ Control de incidencias y reparaciones
- ✅ Historial de mantenimiento por equipo
- ✅ Consulta del estado de los equipos

---

## 💻 Tecnologías utilizadas

- PHP  
- MySQL  
- HTML  
- CSS  
- JavaScript
